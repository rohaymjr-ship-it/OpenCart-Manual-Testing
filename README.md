# OpenCart Demo Testing Project

## Overview

This project demonstrates my Manual Testing skills using the OpenCart Demo website.

The project includes test scenarios, test cases, bug reporting, and test execution documentation covering the main functionalities of the website.

## Website Under Test

https://demo.opencart.com

## Scope

The following modules were tested:

- User Registration
- Login
- Product Search
- Product Details
- Shopping Cart
- Checkout
- Navigation
- UI Validation

## Testing Types

- Functional Testing
- Positive Testing
- Negative Testing
- UI Testing
- Smoke Testing
- Regression Testing

### Test Cases
